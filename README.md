📊 Customer Sales Analysis using EDA

📌 Overview

This project performs Exploratory Data Analysis (EDA) on a customer sales dataset to uncover meaningful insights about customer behavior, purchasing patterns, and business opportunities.

The analysis helps in identifying target customers, high-performing categories, and regional trends to support better business decisions.

🎯 Objectives
Perform data cleaning and preprocessing
Analyze customer demographics and behavior
Identify top-performing product categories
Generate insights to improve sales and customer experience

🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn

🔍 Steps Performed

1. Data Cleaning & Preprocessing
Removed null values and duplicates
Converted data types for accurate analysis
Cleaned inconsistent data entries
2. Exploratory Data Analysis
Analyzed customer demographics (age, gender, marital status)
Performed grouping and aggregation using Pandas
Identified trends across states, occupations, and categories
3. Data Visualization
Created bar charts, count plots, and heatmaps
Visualized relationships between different variables
Highlighted top-performing segments
